## 如何开始

```bash
pnpm dlx nuxi@latest init -t github:moujinet/modoc-starter
```

---

## 如何参与

1. 克隆仓库 `git clone https://github.com/moujinet/modoc.git`
2. 安装依赖 `pnpm install`
3. 运行 `pnpm run dev`
4. 提前运行 `pnpm run lint:fix` 与 `pnpm run typecheck`.
